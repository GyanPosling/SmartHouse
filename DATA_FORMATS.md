# Форматы данных

## users.txt
Формат: `ID|USERNAME|PASSWORD`

Пример:
```
1|admin|password123
2|user1|mypassword
```

## devices.txt
Формат: `ID|DEVICE_NAME|LOCATION|MODE|POWER_LEVEL`

Где:
- ID - целое число (уникальный идентификатор)
- DEVICE_NAME - имя устройства
- LOCATION - местоположение устройства
- MODE - режим работы (0=Автоматический, 1=Ручной, 2=Выключен)
- POWER_LEVEL - уровень мощности (целое число

Пример:
```
1|Кондиционер гостиная|Гостиная|0|5
2|Обогреватель спальня|Спальня|0|3
3|Увлажнитель кухня|Кухня|1|2
```

## sensors.txt
Формат: `ID|SENSOR_NAME|VALUE|UNIT`

Где:
- ID - целое число (уникальный идентификатор)
- SENSOR_NAME - имя сенсора
- VALUE - значение (число с плавающей точкой)
- UNIT - единица измерения

Пример:
```
1|Temperature|22.5|C
2|Humidity|45.0|%
```
